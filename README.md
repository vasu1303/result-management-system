This repository contains the code for a basic frontend only Result Management System.

Repository Link : [Click here](https://github.com/vasu1303/result-management-system)

### Stack Used:
- HTML
- CSS (Bootstrap)
- JavaScript

### Storage:
- Local Storage in the browser.

### Step to run
- Clone this repository
- Open it inside an ide of your choice
- Open the `index.html` file and start the live server
- Application should be live now at `http://localhost:3000`
